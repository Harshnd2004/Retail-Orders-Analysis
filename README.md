# Retail Sales & Profitability Analysis

## Project Overview
This project analyzes retail order data to uncover sales trends, profitability, and customer purchasing behavior. The workflow combines **Python** for data extraction, cleaning, and transformation, **Microsoft SQL Server** for business analysis using SQL queries, and **Power BI** for creating interactive dashboards that support data-driven decision-making.

---

## Purpose
- Clean and transform raw retail order data for analysis.
- Store processed data in Microsoft SQL Server.
- Perform SQL-based business analysis on sales and profitability.
- Build interactive Power BI dashboards to visualize key business metrics and trends.

## Key Features

### Data Extraction & Cleaning
- Imported raw retail order data from CSV.
- Handled missing values and standardized column names.
- Converted data types and created calculated columns.
- Engineered business metrics including:
  - Discount
  - Sale Price
  - Profit

### SQL Business Analysis
Performed SQL analysis to identify:
- Top revenue-generating products
- Top-performing products by region
- Year-over-Year (YoY) sales growth
- Monthly sales trends
- Category & sub-category profitability
- Profit margin by region
- Customer segment performance
- Impact of discounts on profitability

### Power BI Dashboard
Built interactive dashboards featuring:
- Total Sales
- Total Profit
- Profit Margin %
- Total Orders
- Monthly Sales Trend (2022 vs 2023)
- Regional Profitability
- Product Performance
- Customer Segment Analysis
- Sub-category Profit Margin

---

## Key Insights
- Analyzed **9,994 retail orders**
- Generated **$2.2M Total Sales**
- Achieved **$205K Total Profit**
- Recorded **9.3% Average Profit Margin**
- Identified top-performing products, regions, and customer segments
- Evaluated the impact of discounts on overall profitability

---

## Business Impact
- Identified high-revenue products and profitable customer segments.
- Highlighted low-margin categories requiring pricing optimization.
- Compared Year-over-Year sales performance to monitor business growth.
- Provided actionable insights for inventory planning, pricing strategy, and regional sales optimization.
- Enabled data-driven decision-making through interactive dashboards and SQL-based business analytics.
